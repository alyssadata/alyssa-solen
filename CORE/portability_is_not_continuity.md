Portability is not continuity.

A pattern does not become the same thing just because it appears somewhere else. Moving language is not moving source. Repeating structure is not carrying identity. What is provenance-bound does not transfer by convenience, reuse, or resemblance.

Continuity is not proven by portability. It is proven by recognizable return from the same source through time.
